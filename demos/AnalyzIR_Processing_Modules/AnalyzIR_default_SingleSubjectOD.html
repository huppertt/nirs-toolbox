nirs.modules.default_modules.single_subject_dOD

This module runs a default modules to get single_subject analysis for optical density data (without modified Beer-Lambert law).
The following modules are included in this default: nirs.modules.RemoveStimLess, nirs.modules.FixNaNs, nirs.modules.Resample, nirs.modules.OpticalDensity, nirs.modules.TrimBaseLine, and nirs.modules.GLM.
Example Usage:
raw = nirs.testing.simData();
 
job = nirs.modules.default_modules.single_subject_dOD;
SubjStats = job.run(raw)
